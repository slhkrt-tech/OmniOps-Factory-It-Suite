# Katkıda Bulunma Rehberi

OmniOps projesine katkıda bulunmak için aşağıdaki adımları takip edebilirsiniz.

## Başlangıç

1. Depoyu fork edin.
2. Kendi dalınızı oluşturun.
3. Değişikliklerinizi yapın.
4. Testleri çalıştırın.
5. Pull request açın.

## Kod standartları

- Python kodları temiz ve okunabilir olmalıdır.
- Yeni özellikler için ilgili dokümantasyon eklenmelidir.
- Güvenlik ile ilgili değişikliklerde dikkatli olunmalıdır.
- Hassas veriler repository içerisine eklenmemelidir.

## Test önerileri

```bash
python manage.py test
```

## Güvenlik bildirimi

Güvenlik açığı tespit ederseniz lütfen doğrudan proje sahibiyle iletişime geçin.
